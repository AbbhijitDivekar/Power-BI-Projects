# India AQI Analysis (2015–2025)

This project analyzes **Air Quality Index (AQI)** data across **Indian states and cities**
from **January 2015 to June 2025** using **Power BI**.

## Key Highlights
- State-wise and city-wise AQI comparison
- Long-term AQI trends
- AQI categories (Good to Severe)
- Pollutant analysis (PM2.5, PM10, NO2, SO2, CO, O3)
- Drill-down from State → City → Date

## Tools & Technologies
- Power BI Desktop
- Power BI Service
- DAX
- Data Modeling
- Azure Maps
